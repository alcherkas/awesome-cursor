# Awesome Cursor

Structured curriculum to master Cursor IDE. Rendered as a site via [Docsify](https://docsify.js.org).

## GitHub Pages

1. Push this repo to GitHub.
2. **Settings** → **Pages** → **Build and deployment**:
   - Source: **Deploy from a branch**
   - Branch: **main** (or your default), folder: **/ (root)**
3. Save. The site will be at `https://<username>.github.io/awesome-cursor/` (or your custom domain).

To show an “Edit on GitHub” link, set `repo: 'your-username/awesome-cursor'` in `index.html`.
